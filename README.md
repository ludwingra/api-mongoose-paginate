# api-mongoose-paginate

### Plugin: mongoose-paginate-v2

### npm i
### npm run dev

#### Routs
```
// POST
http://localhost:3000/products

// GET
  http://localhost:3000/products
  // Query paginate:
  http://localhost:3000/products?page=1&limit=10

// GET BY ID
  http://localhost:3000/products/:id

// PUT
  http://localhost:3000/products/:id

// DELETE
  http://localhost:3000/products/:id

```
